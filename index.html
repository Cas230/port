My Programming Journey

How I fell in love with coding

Hi! Jomari Casiño, a student at STI Alaminos, where my passion for programming began.

Starting with Java

When I began my college journey, Java was the first programming language I encountered. I was thrilled and full of excitement—Java opened the door to the world of coding for me! Although I initially found it difficult, particularly when it came to understanding functions, those struggles motivated me to put in more effort. Despite the challenges, I truly enjoyed the experience.

First Knowing About C#

After that, I was introduced to C#. Similar to Java, it came with its challenges, but I found it fascinating. I could envision myself creating real-world applications with it, particularly for Android, and that idea really inspired me. It quickly became one of my favorite languages to dive into.

Then Python Came

Next, I encountered Python—a language known for its simplicity and power. I personally found it more straightforward than Java or C#, which made it easier to grasp. Interestingly though, whenever I concentrated on Python, I noticed myself forgetting concepts from Java and CSS! Even so, learning Python was a valuable and enjoyable experience.

Enjoying with Android Studio

Working with Android Studio was a fun and rewarding experience. It gave me the opportunity to apply what I had learned in Java and C# to build actual mobile apps. Seeing my ideas come to life on a screen was exciting and deeply satisfying—it made all the effort feel worth it and sparked my passion for app development even more.
